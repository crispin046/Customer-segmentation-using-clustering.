# Customer Segmentation using RFM & AOV Analysis

## Overview

This project performs a comprehensive **customer segmentation analysis** using the **Recency, Frequency, Monetary (RFM)** model enhanced with **Average Order Value (AOV)** to identify distinct customer groups within an online retail dataset.

The objective is to generate **actionable insights for targeted marketing strategies**, helping businesses improve **customer engagement, retention, and overall profitability**.

---

## Key Features

- **Data Cleaning and Preprocessing**
  - Handling missing values
  - Data type conversions
  - Removal of invalid transactions

- **RFM Metric Calculation**
  - Recency
  - Frequency
  - Monetary value

- **Log Transformation and Scaling**
  - Reducing skewness in the data
  - Normalizing feature scales for clustering

- **Elbow Method**
  - Determining the optimal number of clusters for K-Means

- **K-Means Clustering**
  - Segmenting customers based on behavioral patterns

- **AOV Integration**
  - Extending RFM by incorporating **Average Order Value (AOV)**

- **Segment Profiling & Visualization**
  - Understanding cluster characteristics through visual analytics

- **Marketing Strategy Recommendations**
  - Data-driven strategies tailored to each customer segment

---

## Dataset

The analysis uses the **Online Retail.xlsx** dataset, which contains transactional data for a UK-based online retail store.

### Dataset Attributes

- CustomerID
- InvoiceNo
- StockCode
- Description
- Quantity
- UnitPrice
- InvoiceDate
- Country

The dataset represents **transactional purchase behavior** of customers in an e-commerce environment.

---

## Methodology

### 1. Data Loading & Initial Inspection

The dataset is loaded into a **Pandas DataFrame**, followed by an inspection of:

- Dataset shape
- First rows of data
- Missing values
- Data types

---

### 2. Data Cleaning

Data quality improvements include:

- Removing rows with missing **CustomerID**
- Removing cancelled orders (InvoiceNo starting with **"C"**)
- Removing transactions with:
  - Zero or negative quantities
  - Zero or negative unit prices
- Creating a **Revenue column**

```python
Revenue = Quantity * UnitPrice
```

## 3. RFM Calculation

RFM metrics are computed for each customer:

### Recency
- Number of days since the customer's last purchase.

### Frequency
- Number of unique purchase transactions made by the customer.

### Monetary
- Total revenue generated by the customer.

---

## 4. Data Preprocessing for Clustering

To prepare the data for clustering:

- **Log transformation (`log1p`)** is applied to reduce skewness in the distribution of RFM values.
- **StandardScaler** is used to normalize the variables so they contribute equally during clustering.

---

## 5. Optimal Cluster Determination

The **Elbow Method** is applied to determine the optimal number of clusters for segmentation.

For this dataset, the optimal number of clusters is:

**4 clusters**

---

## 6. K-Means Clustering

Customers are segmented into **4 clusters** using the **K-Means clustering algorithm** based on the preprocessed RFM values.

This allows grouping customers according to their purchasing behavior.

---

## 7. AOV Integration

To enhance the segmentation model, **Average Order Value (AOV)** is introduced.

```python
AOV = Monetary / Frequency
```
- The clustering process is repeated using the following variables:

* Recency

* Frequency

* Monetary

* AOV

- All variables are again log-transformed and scaled before running the clustering algorithm.

### 8. Segment Analysis & Visualization

- Customer segments are analyzed by calculating the average RFM and AOV values for each cluster.

Visualization techniques include:

* Bar plots

* Cluster comparison charts

* Standardized feature profiles

These visualizations help interpret behavioral differences between customer segments.
# Key Findings

---

The enhanced **RFM + AOV analysis** revealed **four distinct customer segments**.

## 1. Champions (Cluster 0)

- **13.1 purchases**
- **$8,128.5 total spend**
- **$723.5 AOV**
- **Purchased 20.4 days ago**

### Profile
Highly engaged customers who purchase frequently and spend large amounts.  
They represent the **most valuable and loyal customer segment**.

---

## 2. Big Spenders (Cluster 1)
*(Infrequent but High AOV)*

- **1.8 purchases**
- **$999.0 total spend**
- **$615.3 AOV**
- **Purchased 140.7 days ago**

### Profile
Customers who make **large purchases but do so infrequently**.  
They have **high revenue potential if their purchase frequency increases**.

---

## 3. Regular Shoppers (Cluster 2)
*(Frequent but Low AOV)*

- **3.8 purchases**
- **$994.2 total spend**
- **$275.6 AOV**
- **Purchased 26.5 days ago**

### Profile
Customers who shop **relatively frequently** but tend to make **smaller purchases per order**.

**Opportunity:** Increase their **Average Order Value** through upselling and cross-selling strategies.

---

## 4. At-Risk / Lapsed Customers (Cluster 3)

- **1.5 purchases**
- **$206.0 total spend**
- **$145.4 AOV**
- **Purchased 156.9 days ago**

### Profile
Customers who have **not purchased recently** and demonstrate **low frequency and low spending behavior**, indicating a **high risk of churn**.
